# Yêu cầu phi chức năng

| Tiêu chí | Mô tả |
| :--- | :--- |
| Performance | Thời gian phản hồi của hệ thống ở mức 2-3 giây, không yêu cầu quá nhanh. |
| Security | Hệ thống phải đảm bảo tính bảo mật, bao gồm việc mã hóa mật khẩu người dùng và phân quyền truy cập chặt chẽ. Phân quyền theo từng người dùng. |
| UX/UI | Giao diện người dùng phải thân thiện, dễ sử dụng |
| Compatibility | Hệ thống phải tương thích với nhiều thiết bị khác nhau, bao gồm cả máy tính để bàn, máy tính xách tay, máy tính bảng và điện thoại thông minh. |
| Reliability | Đảm bảo hệ thống có khả năng chịu lỗi. |
| Maintainability | Hệ thống phải dễ bảo trì và bổ sung tính năng mới trong tương lai. |
| Portability | Hệ thống phải có khả năng chạy trên nhiều nền tảng khác nhau. |
