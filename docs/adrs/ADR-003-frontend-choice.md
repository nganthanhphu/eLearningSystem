# ADR 003: Lựa chọn ReactJs cho Frontend
## Trạng thái
Accepted
## Bối cảnh
Hệ thống học trực tuyến yêu cầu các chức năng tương tác như xem video bài giảng, làm quiz, theo dõi tiến độ học và quản lý khóa học. Các chức năng này cần giao diện có khả năng cập nhật dữ liệu từ backend thông qua API và đảm bảo trải nghiệm người dùng mượt mà. Ngoài ra hệ thống có thể linh hoạt mở rộng chức năng khi cần
## Quyết định
Sử dụng ReactJS cho Frontend
## Lý do
1. Hiệu năng tốt nhờ Virtual Dom 
2. Phù hợp với giao diện có nhiều tương tác
3. Kiến trúc component giúp tái sử dụng giao diện
4. Hệ sinh thái,cộng đồng lớn
## Hệ quả
1. Tích cực: 
+ Trải nghiệm user tốt hơn
+ Hệ thống dễ mở rộng thêm chức năng 
+ Dễ bảo trì
2. Tiêu cực:
+ Cần kết hợp với các thư viện khác để xây dựng hệ thống lớn hoàn chỉnh có kiến ​​trúc tốt
